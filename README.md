# hoge

# 概要
最新論文を追実装してみようという試み

# 動作確認環境
OS X 10.10.5
openFrameworks v0.9.3

# 使い方とか
現状はopenFrameworksのプロジェクトなので、このリポジトリを丸ごとapps/myApps以下においてください。

サブモジュールを更新する必要があるので、以下のコマンドを打っておいてください。

    git submodule update --init --recursive
